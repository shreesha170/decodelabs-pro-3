# TaskFlow Dashboard

## Overview

TaskFlow Dashboard is a responsive and interactive task management web application built using HTML5, CSS3, and JavaScript. It allows users to add tasks, delete tasks, and switch between light and dark themes through a simple and user-friendly interface.

## Features

### Task Management

* Add new tasks dynamically
* Input validation for empty tasks
* Delete tasks individually
* Clean and responsive task card layout

### Dark Mode Toggle

* Switch between light and dark themes
* Smooth theme transitions
* Improved user experience and accessibility

### Responsive Design

* Mobile-friendly layout
* Tablet and desktop support
* Responsive navigation bar
* Flexible form layout using Flexbox

### User Experience

* Smooth scrolling
* Hover effects on buttons and navigation links
* Animated task cards using CSS keyframes
* Modern and clean interface

## Technologies Used

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Add tasks and explore the dark mode feature.

## Responsive Breakpoints

* Desktop: Above 768px
* Tablet: Up to 768px
* Mobile: Optimized layout with stacked navigation and form elements
